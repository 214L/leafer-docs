# disabled

禁用状态，暂时不支持 Group。

需安装 [交互状态 插件](/plugin/in/state/index.md) 才能使用。

## 关键属性

### disabled： `boolean`

是否禁用。

### disabledStyle: [`IUIInputData`](/api/interfaces/IUIInputData.md)

元素 `disabled` 设为 true 时的禁用样式， `disabled` 设为 false 后自动还原。
