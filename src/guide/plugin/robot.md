# Robot

Robot 元素，类似游戏中的精灵，可用于制作游戏动作、组态动画。

通过加载图片序列帧进行显示、播放，产生各种固定、动态动作。

::: tip 注意事项
需安装 [Robot 插件](/plugin/in/robot/) 才能使用。

:::
